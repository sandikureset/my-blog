## Apa itu GitHub Pages?

GitHub Pages adalah layanan hosting **gratis** dari GitHub yang bisa menampilkan website statis seperti HTML, CSS, JavaScript, bahkan Markdown.

Kamu hanya perlu membuat repository publik, upload file website kamu, dan aktifkan Pages di pengaturannya.

---

## Kenapa pakai Markdown?

Karena *Markdown itu simpel banget!*  
Cukup pakai tanda bintang `*`, pagar `#`, dan garis `---` kamu sudah bisa buat struktur artikel seperti ini.

---

## Yuk Coba Sendiri

1. Buat akun di [GitHub.com](https://github.com)
2. Buat repo baru bernama `my-blog`
3. Upload ketiga file ini (`index.html`, `style.css`, `artikel1.md`)
4. Aktifkan **GitHub Pages**
5. Buka hasilnya di:  
   `https://username.github.io/my-blog/`
